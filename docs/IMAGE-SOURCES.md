# 실사 이미지 채우기 가이드 (초상·장비)

앱은 **파일만 떨궈 넣으면** 카드에 실제 사진이 뜨도록 설계돼 있습니다.
이미지가 없으면 자동으로 도식 초상/실루엣으로 폴백하고, 로드에 성공한 경우에만
출처·라이선스 각주가 표시됩니다. (`src/components/Panel/PersonCard.tsx`,
`EquipmentCard.tsx`의 `onError` 폴백 참고)

## 현재 배선 상태 (2026-07 기준)

외부망 리서치로 확인된 **공개/자유 라이선스** 이미지를 위키미디어 커먼즈 직접
URL로 연결했습니다(핫링크). 배포 사이트(사용자 브라우저)는 정상 로드되며, 로드
실패 시 도식 초상/실루엣으로 자동 폴백합니다.

- **초상 연결됨(5)**: 심일 · 김종오 · 김용배 · 민병권 · 조달진
- **도식 유지(3)**: 임부택 · 정오경 · 함병선 — 신원 확인된 자유 라이선스 초상을
  아직 찾지 못함(오식별 방지 위해 실사 미적용)
- **장비 연결됨(6, 전부)**: SU-76 · T-34-85 · BA-64 · M3 105mm · M1 57mm · M1 81mm

### 권장: 셀프호스팅으로 전환(오프라인·안정성)

현재는 `upload.wikimedia.org` 직접 링크라 오프라인(PWA) 캐시가 안 되고, 원격
호스트에 의존합니다. 더 견고하게 하려면 각 이미지를 내려받아
`public/portraits/<id>.jpg` · `public/equipment/<id>.jpg` 로 저장한 뒤,
`src/data/people.ts` · `equipment.ts`의 `photo.src`를 로컬 경로로 바꾸세요.
(라이선스가 CC BY / CC BY-SA인 항목은 `credit`·`license` 표기를 유지해야 합니다.)

> ⚠️ 이 세션의 네트워크 정책상 외부 이미지 호스트가 차단돼 Claude가 파일을 직접
> 내려받지 못합니다. 셀프호스팅 전환은 개방망 세션/로컬에서 진행하거나, 파일을
> 주시면 제가 경로를 바꿔 배선하겠습니다.

## 채우는 방법 (2단계)

### 1) 이미지 파일을 아래 경로에 저장

```
public/portraits/<id>.jpg     # 인물 초상 (세로 3:4 권장, 최소 240×320)
public/equipment/<id>.jpg     # 장비 사진 (가로, 투명/단색 배경이면 더 좋음)
```

### 2) 데이터에 `photo` 항목 추가 (출처·라이선스 확인 후)

**인물** — `src/data/people.ts`의 `personMeta[id]`에 `photo` 추가:

```ts
'kim-jongo': {
  hanja: '金鍾五',
  medals: ['태극무공훈장'],
  photo: {
    src: '/portraits/kim-jongo.jpg',
    credit: '국가기록원',          // 실제 소장처로 교체
    license: 'KOGL 제1유형',       // 실제 라이선스로 교체
    sourceUrl: 'https://...',      // 원본 페이지(선택)
    note: '1950년대',              // (선택)
  },
},
```

**장비** — `src/data/equipment.ts`의 해당 항목에 `photo` 추가:

```ts
{
  id: 't-34',
  // ...기존 필드...
  photo: {
    src: '/equipment/t-34.jpg',
    credit: 'Wikimedia Commons',
    license: 'Public Domain',
    sourceUrl: 'https://commons.wikimedia.org/wiki/File:...',
  },
},
```
